# qiita_tag_items

## 使用環境
### 動作確認済み
PC : MacBookPro(2020)  
OS : macOS 11.2.3  

## 使い方
1. get_tags_*.shを編集

収集したいタグ名に変更  
例：リファクタリングをアンチパターンに変更

```get_tags_*.sh
# TAG_ID='リファクタリング'
TAG_ID='アンチパターン'
```

2. get_tags_*.shをコール
3. ranking_*.jsonを確認
